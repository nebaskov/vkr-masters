Backbone - основная часть модели, выполняющая главное преобразование
последовательности токенов\\[0.5em]
BPN (bits per nucleotide) - количество бит на нуклеотид, используемое
как языковая метрика качества\\[0.5em]
Compression ratio (CR) - коэффициент сжатия последовательности,
вычисляемый как $1-N/L$\\[0.5em]
caQTL - вариант генома, статистически ассоциированный с изменением
доступности хроматина\\[0.5em]
cCREs - candidate cis-regulatory elements, кандидатные
цис-регуляторные элементы по аннотациям ENCODE\\[0.5em]
ClinVar - база данных клинически интерпретированных генетических
вариантов\\[0.5em]
dsQTL - вариант генома, ассоциированный с изменением чувствительности
к DNase I\\[0.5em]
EPDnew - Eukaryotic Promoter Database, база аннотированных
эукариотических промоторов\\[0.5em]
GENCODE - референсная аннотация генов и транскриптов для геномов
человека и мыши\\[0.5em]
JASPAR - открытая база мотивов связывания транскрипционных факторов
(PWM-профили)\\[0.5em]
LINE - long interspersed nuclear elements, класс длинных
повторяющихся ретротранспозонов\\[0.5em]
Hard-forward / soft-backward - схема обучения, в которой прямой
проход использует жёсткие решения, а обратный проход использует
мягкую релаксацию\\[0.5em]
k-мер - подпоследовательность ДНК фиксированной длины $k$\\[0.5em]
NTP (next nucleotide prediction) - задача предсказания следующего
нуклеотида по предшествующему контексту\\[0.5em]
phastCons - phylogenetic hidden Markov model conservation scores,
количественная оценка эволюционной консервативности нуклеотидных
позиций\\[0.5em]
SINE - short interspersed nuclear elements, класс коротких
повторяющихся ретротранспозонов\\[0.5em]
Alu - наиболее распространённое у приматов семейство SINE-повторов\\[0.5em]
Мотив - короткий консервативный паттерн в ДНК, связанный с
функциональным элементом\\[0.5em]
Промотор - участок ДНК, где инициируется транскрипция гена\\[0.5em]
Энхансер - регуляторный участок ДНК, усиливающий транскрипцию, часто
на удалении от промотора\\[0.5em]
Накопительное состояние токена - дополнительное представление,
агрегирующее историю предыдущих слияний токена\\[0.5em]
Нуклеотид - мономерная единица ДНК (аденин A, тимин T, цитозин C,
гуанин G)\\[0.5em]
Обратный комплемент - последовательность, полученная заменой каждого
нуклеотида комплементарным и разворотом в обратном направлении\\[0.5em]
TFBS - transcription factor binding site, сайт связывания
транскрипционного фактора\\[0.5em]
Спан - непрерывный диапазон позиций в исходной последовательности,
покрытый одним токеном после слияния\\[0.5em]
Token ratio (TR) - отношение числа токенов после сжатия к длине
исходной последовательности, вычисляемое как $N/L$\\[0.5em]
Токенизация - процесс разбиения входной последовательности на
дискретные единицы (токены) для обработки моделью\\[0.5em]
Токенизатор - модуль, преобразующий входную последовательность в
последовательность токенов\\[0.5em]
Функция потерь - целевая функция, минимизируемая в процессе обучения модели
